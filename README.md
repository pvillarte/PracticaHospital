# PracticaHospital

A hospital management system built in C++ to practice object-oriented programming concepts.

Manages patients, doctors and appointments through a console interface with a custom pagination system for easy navigation.

## Concepts applied

- OOP: classes, inheritance, encapsulation, polymorphism
- Custom pagination for console navigation
- Data relationships between entities (patients, doctors, appointments)

## Build

Open the solution in Visual Studio and build the `PracticaHospital` project.

Requires: Visual Studio with C++ workload.
